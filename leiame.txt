GIT INIT
Inicia o controle de dimencionamento

GIT ADD <ARQUIVO>
Adiciona o arquivo/alteração na estage alteração

GIT COMMIT -M "SEU_TEXTO"
Adicionando comentario do conteudo da estage alteração

GIT STATUS
Valida as alterações no projeto

GIT LOG
Mostra o historico da BRANCH 

GIT CHECKOUT -B <nome_branch>
Ele cria uma nova "branch"
